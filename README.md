# OpenClaw Hardware Integration Monitor

## Overview
This repository contains a real-time monitoring dashboard for OpenClaw hardware integration projects across multiple platforms including ESP32, Arduino, Raspberry Pi, and NVIDIA Jetson.

The dashboard tracks:
- GitHub repositories with OpenClaw integration
- Star counts and activity metrics  
- Hardware platform compatibility
- Product links and documentation
- Real-time updates and project status

## Key Projects

### Core Ecosystem
- **Official OpenClaw**: 251k+ stars - The main AI assistant framework
- **Picoclaw (Sipeed)**: 21.8k stars - Tiny, fast, deployable anywhere
- **MimiClaw**: 3.7k stars - Run OpenClaw on $5 chips

### Hardware Platforms
- **ESP32/XIAO**: Multiple IoT agent implementations
- **Raspberry Pi**: Full system installations and lightweight agents  
- **NVIDIA Jetson**: AI compute platform integrations
- **Arduino**: Code generation and maker skills

### Specialized Projects
- **WireClaw**: ESP32 AI agent with offline rule engine
- **OpenClawgotchi**: AI Tamagotchi with E-Ink display
- **Monagotchi**: ESP32 + memecoin integration
- **Reachy Mini**: Robotics integration

## Usage
Visit the live dashboard at: https://256ericpan.github.io/clawhardware/

Features:
- Filter by application category (Robotics, IoT, Embedded, Maker, AI Compute)
- Filter by technology platform (ESP32, Arduino, Raspberry Pi, Jetson)
- Search by project name or description
- Sort by GitHub stars (descending)
- View product links and hardware specifications

## Data Sources
- GitHub API for repository metadata
- Manual curation of hardware product links
- Community project discovery through keyword searches

## Contributing
New projects are automatically discovered through regular monitoring. If you have a project that should be included, please ensure it has "openclaw" in the repository name or description.
